# 📝 Документация по заполнению данных для Pixel Dash Demon List

Этот файл содержит инструкции по правильному добавлению уровней и рекордов в конфигурационные файлы `list.json` и `verify.json`.

---

## 📂 1. Файл `data/list.json` (Список демонов)

Этот файл отвечает за вывод карточек уровней на главной странице. Уровни отображаются в том порядке, в котором они записаны в файле (первый в списке получит `#1`, второй — `#2` и так далее).

### 📋 Обозначения сложностей (`difficulty`)
Для корректного отображения иконки демона, в поле `difficulty` необходимо передавать число от **1** до **5**:
* `1` — **Easy Demon** 🟢
* `2` — **Medium Demon** 🟡
* `3` — **Hard Demon** 🟠
* `4` — **Insane Demon** 🔴
* `5` — **Extreme Demon** 🟣

### ⌨️ Шаблон для копирования:
```json
{
  "id": 1,
  "name": "Название Уровня",
  "level_id": "ID_из_игры",
  "creator": "Создатель_уровня",
  "publisher": "Кто_верифицировал",
  "difficulty": 5
}
```
